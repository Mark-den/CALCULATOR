# Calculator

A simple web-based calculator implemented with plain HTML, CSS and JavaScript.

## Files

- [index.html](index.html): HTML layout and buttons.
- [script.js](script.js): Calculator logic and event handling.
- [style.css](style.css): Styles and layout.

## Usage

 Open `index.html` in your web browser.


## Features

- Basic arithmetic (+, -, ×, ÷)
- Clear and backspace
- Keyboard support

## Notes

- This is a minimal frontend project — no build step required.
- Feel free to add tests, accessibility improvements, or styling tweaks.
